# Contoh Penggunaan Socket.IO dengan Bootstrap

Proyek ini mendemonstrasikan cara menggunakan Socket.IO untuk komunikasi real-time antara server Node.js dan klien web, dengan tampilan antarmuka yang diatur menggunakan Bootstrap 5.

## Fitur

- Koneksi WebSocket menggunakan Socket.IO.
- Pengiriman dan penerimaan pesan secara real-time.
- Tampilan antarmuka yang responsif dan modern menggunakan Bootstrap.

## Instalasi

1. Clone repository ini.
2. Navigasi ke direktori proyek: `cd nama-repo-anda`
3. Instal dependensi: `npm install`

## Menjalankan

Jalankan server menggunakan perintah:
`npm start`

Buka browser Anda dan kunjungi `http://localhost:3000`.

## Dependensi

- Node.js
- npm atau yarn
- Bootstrap 5 (diintegrasikan via CDN)
